### Vircon32 について知るには？

手短に説明すると、 Vircon32 は 32-bit 仮想ゲーム機です。詳細情報は公式ウェブサイト [www.vircon32.com](http://www.vircon32.com/ "Vircon32 ホームページ") を参照してください。公式ウェブサイトにはゲームを遊ぶのと作るために知る必要のある情報を掲載してあります。
